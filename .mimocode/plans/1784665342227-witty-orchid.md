## Architecture: Single-Page Site with 4 Sections + Navbar + Footer

### Files to Modify
1. **`src/main.ts`** — Replace boilerplate with full site HTML + scroll animations + mobile menu logic
2. **`src/style.css`** — Already has theme, fonts, and animations; minor additions only
3. **`index.html`** — Already correct, no changes needed

### Files to Delete
- `src/counter.ts` (unused template file)

### Sections (matching user menu request)
1. **Navbar** — Sticky top, logo "Zona Ziette" (SVG inline with gold gradient Z), nav links: Nosotros | SFTRACKER | Nuestra Filosofía | Contáctanos. Mobile hamburger menu.
2. **Hero** — Full-screen gradient background (brown-dark to brown), large headline about posmarketing software, CTA button
3. **Nosotros** — About the company: what Zona Ziette does (posmarketing, promotoría, software solutions)
4. **SFTRACKER** — Product showcase with 6 feature cards:
   - Evidencias (photo evidence capture)
   - Check-in (location check-in)
   - Formularios (digital forms)
   - Ubicación (GPS tracking)
   - Participación de Mercado (market presence analytics)
   - Dashboard/reportes
5. **Nuestra Filosofía** — Company values/mission: innovation, data-driven, field intelligence
6. **Contáctanos** — Contact form (name, email, company, message) + contact info
7. **Footer** — Logo, copyright, social links placeholder

### Technical Details
- All HTML rendered via `document.querySelector('#app').innerHTML` template literal in `main.ts`
- Scroll animations via IntersectionObserver on `.fade-up` elements
- Mobile menu via JS toggle on hamburger button
- Inline SVG for the "Z" logo with gradient matching the brand
- Lucide icons used for feature cards (Camera, MapPin, FileText, Navigation, BarChart3, LayoutDashboard)
- No routing needed — single page with smooth scroll to sections
